# X UI Clone

A responsive X (Twitter) inspired frontend UI clone built using HTML and TailwindCSS.

This project was created to practice modern frontend layout techniques, Tailwind utility classes, and responsive design.

⚠️ This is an educational project only and is not affiliated with X.

## Tech Stack

- HTML  
- TailwindCSS  

## Features

- X-style feed layout  
- Circular profile avatars  
- Post UI components  
- Responsive design  
- Clean Tailwind utility-based styling
